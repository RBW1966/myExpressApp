Technologies used in the myExpressApp website

This web page is the public home page for the myExpressApp web site. The purpose is to showcase technologies used in the development of this site and provide a link for registered users to log in to the web apps.

  • Amazon Web Services (AWS)

    The AWS Route 53 service was used to register the robweed.io domain and to configure a public hosted zone to route internet traffic to the web server. The web server is an AWS Elastic Cloud Computing (EC2) instance hosted in Northern California. The AWS Simple Storage Service (S3) is used for file storage.
